Overview

Synopsis

This Class modifies overwrites the default Mysql Class. It changes the behaviour, so that each request from the database is returning a UTF-8 encoded answer.

def each_hash_utf8(&block)
  each_hash_orig do |row|
    row.each {|k, v| row[k] = encode(v) }
    yield(row)
  end
end

def each_utf8(&block)
  each_orig do |row|
    yield row.map {|col| encode(col) }
  end
end
